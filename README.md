# Ale e Nebo - Wedding Website

Wedding website for Ale e Nebo - 19/09/2026

## Setup Instructions

### Adding Your Photo

1. Add your couple photo to the `images` folder
2. Name it `couple-photo.jpg` (or update the CSS file if you use a different name/format)
3. Recommended size: 1920x1080 pixels or larger for best quality

### Customizing Content

Edit `index.html` to update:
- Church name and address
- Ceremony time
- Reception venue name and address
- Reception time

### Opening the Website

Simply open `index.html` in your web browser to view the website.

## Features

- Responsive design (works on mobile and desktop)
- Smooth scrolling between sections
- Fade-in animations
- Parallax effect on hero image
- Clean, elegant design
